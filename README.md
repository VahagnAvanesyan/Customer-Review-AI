# Customer-Review-AI
# 🧠 Customer Review AI

An AI-powered web application for analyzing and understanding customer reviews using machine learning / NLP.

---

## 🚀 Project Overview

Customer Review AI is a web-based tool that allows users to input reviews and receive intelligent analysis such as sentiment detection, feedback classification, and insights.

The goal of this project is to demonstrate how AI can be used to improve business decision-making by analyzing customer feedback.

## ▶️ How to Run

```bash
# 1. Clone repository
git clone https://github.com/VahagnAvanesyan/Customer-Review-AI.git

# 2. Go to project folder
cd Customer-Review-AI

# 3. Create virtual environment
python -m venv venv

# 4. Activate it
# Windows:
venv\Scripts\activate

# 5. Install dependencies
pip install -r requirements.txt

# 6. Run project
python app.py

```
