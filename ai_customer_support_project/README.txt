# AI Customer Support Assistant Project

Полный готовый проект веб-приложения на Flask с интеграцией ML-модели и базы данных SQLite.

## Порядок запуска:

1. Распакуйте архив в пустую рабочую папку в VS Code.
2. Поместите ваш файл `Dataset (3).csv` внутрь папки `data/` (если хотите обучить модель заново на вашем файле).
3. Инициализируйте пустую базу данных:
   python init_db.py

4. Обучите модель и экспортируйте .pkl файлы:
   python train_pipeline.py

5. Запустите Flask-сервер:
   python app.py

6. Откройте в браузере: http://127.0.0.1:5000
